# Competency Framework

## Python/Go
Weight: 0.35
- Score 10: Expert production engineering in Python or Go at scale
- Score 7: Strong implementation skills with meaningful production ownership
- Score 4: Limited production experience, mostly scripts or coursework

## System Design
Weight: 0.30
- Score 10: Designed large-scale distributed systems with clear tradeoff decisions
- Score 7: Good architecture judgment for mid-scale service design
- Score 4: Basic service design understanding with minimal production evidence

## Databases
Weight: 0.20
- Score 10: Strong relational and distributed data modeling, tuning, and reliability
- Score 7: Solid schema design and query optimization experience
- Score 4: Basic CRUD familiarity with limited production tuning

## Communication
Weight: 0.15
- Score 10: Excellent technical communication with strong cross-functional influence
- Score 7: Clear collaborator with useful design and incident documentation
- Score 4: Limited written and verbal evidence beyond implementation work
