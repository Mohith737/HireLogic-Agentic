# Backend Software Engineer

## Role Summary
Build reliable backend services for internal and customer-facing products. Work spans API design, data storage strategy, performance tuning, and operational resilience for services with sustained production traffic.

## Business Context
- Product lines rely on low-latency APIs and durable data pipelines.
- Engineering teams need strong ownership over service architecture and incident response.
- Clear communication is required across product, infrastructure, and data stakeholders.

## Scope
- Design and maintain backend services in Python or Go.
- Improve service scalability, observability, and fault tolerance.
- Model data access patterns and schema tradeoffs.
- Document interfaces and architecture decisions.
