# Competency Framework

## Python
Weight: 0.30
- Score 10: Expert-level Python, production systems at scale
- Score 7: Proficient, 3+ years, some production experience
- Score 4: Basic, scripts and notebooks only

## Machine Learning
Weight: 0.25
- Score 10: Novel model development, published or production impact
- Score 7: Standard ML pipelines, fine-tuning, deployment
- Score 4: Theoretical knowledge, limited application

## System Design
Weight: 0.20
- Score 10: Designed distributed systems serving millions
- Score 7: Solid architecture knowledge, mid-scale systems
- Score 4: Basic understanding, no production design

## Communication
Weight: 0.15
- Score 10: Led cross-functional projects, clear documentation
- Score 7: Effective team collaborator, decent written communication
- Score 4: Limited evidence of communication beyond individual work

## Research
Weight: 0.10
- Score 10: Published papers, novel contributions
- Score 7: Deep reading, applied research to products
- Score 4: Academic exposure only
